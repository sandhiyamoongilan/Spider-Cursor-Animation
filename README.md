# Spider Cursor Animation | Interactive Web Project

An interactive custom spider cursor animation built using HTML, CSS, and JavaScript, showcasing smooth animations, DOM manipulation, and real-time mouse tracking for engaging web UI effects.

---

## Project Overview

This project demonstrates how to create a dynamic animated cursor that follows mouse movement in real time. The spider animation reacts smoothly to cursor motion, creating an engaging and visually appealing user experience. It is ideal for learning JavaScript event handling, animation logic, and creative UI design.

---

## Key Features

- Real-time mouse tracking animation  
- Smooth spider movement and transitions  
- Clean and responsive UI design  
- Optimized DOM manipulation  
- Lightweight and fast performance  

---

## Tools & Technologies

- HTML5  
- CSS3  
- JavaScript (ES6)  

---

## How to Run

1. Download or clone the repository  
2. Open the `index.html` file in any modern web browser  
3. Move your mouse to view the interactive spider cursor animation  

---

## Project Structure

- `index.html` – Main webpage  
- `style.css` – Styling and layout  
- `script.js` – Animation and cursor logic
